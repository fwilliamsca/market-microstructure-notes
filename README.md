# Market Microstructure & Order Book Dynamics

Personal research notes on high-frequency order book modeling.
Focusing on the correlation between **Order Flow Imbalance (OFI)** and price impact in fragmented crypto markets.

## Key Findings (March 2023)
- OFI is a leading indicator for price changes within the 500ms window.
- Cross-exchange arbitrage opportunities decay exponentially with latency > 200ms.
